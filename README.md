# 🛠️ DigiTools — Premium Digital Tools Marketplace

A modern, responsive e-commerce landing page for browsing and purchasing premium digital
products — AI writing tools, design templates, stock assets, and productivity software — all in
one place.

## 📖 Description

DigiTools is a single-page React application built to showcase a curated catalog of digital
products with a smooth, toast-powered shopping cart experience. Users can browse products,
add them to their cart, review their selection, and proceed to checkout — all without a page
reload.

## ⚙️ Technologies Used

- **React.js** — component-based UI
- **Tailwind CSS** + **DaisyUI** — styling and pre-built UI primitives
- **JavaScript (ES6+)**
- **React-Toastify** — toast notifications for cart actions
- **JSON** — static product data

## ✨ Features

1. **Live cart sync** — Adding a product instantly updates the cart count in the navbar and the
   cart view, powered by a shared React Context.
2. **Toggleable Products/Cart view** — A single section switches between browsing products and
   reviewing the cart, with a friendly empty-cart state when nothing's been added yet.
3. **Toast-driven feedback** — Every cart action (add, remove, checkout) triggers a clear
   react-toastify notification, so the user always knows what just happened.
